# SteamTrade Matcher Userscript

Allows quicker trade offers by automatically adding cards as matched by SteamTrade Matcher on Steam trade offer and SteamTrade Matcher pages.

## Install

[Install](https://github.com/jaredcat/userscripts/raw/refs/heads/main/dist/steam-trade.user.js)

*No userscript manager yet? Install [Violentmonkey](https://violentmonkey.github.io/get-it/) first, then use the link above.*
