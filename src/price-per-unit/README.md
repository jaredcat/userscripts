# Price Per Unit UserScript

This userscript adds price per unit information to product pages on PetSmart.com and enables sorting products by unit price. This helps shoppers make more informed decisions by easily comparing prices across different package sizes.

## Features

- Adds price per unit calculations to product pages
- Adds price per unit to product listing pages
- Adds ability to sort products by unit price (on some sites)
- Supports various units (oz, lbs, count, etc.)

## Supported Sites

Currently supported sites:

- PetSmart.com
- Chewy.com (product pages: price per unit; category/browse pages: "Sort by unit price" in the sort dropdown)
